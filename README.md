# Medical Appointment Scheduling System

## Overview
A Java-based console application for scheduling medical appointments, featuring user management, booking, and reporting. Built with OOP principles and data structures.

## Features
- User registration and login.
- Appointment booking with conflict checking.
- View and cancel appointments.
- Generate reports (e.g., daily schedules).

## Technologies/Tools Used
- Java 11+
- JUnit for testing
- Maven (optional for dependencies)
- Git for version control

## Steps to Install & Run
1. Clone the repo: `git clone <repo-url>`
2. Navigate to `src/main/java`
3. Compile: `javac Main.java`
4. Run: `java Main`
5. Follow console prompts.

## Instructions for Testing
- Run JUnit tests: `mvn test` (if using Maven) or via IDE.
- Manual test: Book an appointment and check for conflicts.
